# 🎙 Emotion Recognition from Speech

## 📌 Project Description

This project predicts human emotions from speech audio using Deep Learning.

The model extracts MFCC (Mel Frequency Cepstral Coefficients) features from speech signals and classifies emotions into different categories.

---

## 🎯 Objective

To identify emotions from voice recordings using a Convolutional Neural Network (CNN).

---

## 📂 Dataset

RAVDESS Emotional Speech Dataset

The dataset contains speech recordings of different emotions.

---

## ⚙ Project Workflow

1. Load Dataset
2. Extract MFCC Features
3. Preprocess Data

5. Train Model
6. Evaluate Accuracy
7. Save Model
8. Predict Emotion
